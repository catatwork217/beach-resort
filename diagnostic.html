❌ Why You See a Blank White Page

The blank white page typically indicates one of these issues:

1. JavaScript Error in Browser

  • Open browser console (F12 → Console tab)
  • Look for red error messages
  • Common errors: undefined variables, network errors, syntax errors

2. React App Not Mounting

  • The div#root element is empty
  • This happens when React fails to render
  • Check console for Uncaught Error messages

3. Contentful Data Structure Mismatch

  • Your code expects home.heroImage but it might be null
  • Your code expects room.images but structure might differ
  • Check the debug logs in console

🔧 How to Debug

Step 1: Open Browser Console

  1. Press F12 or right-click → Inspect
  2. Go to "Console" tab
  3. Refresh the page (F5)
  4. Look for error messages (red text)

Step 2: Check Debug Logs

Look for these console.log messages:

console.log('Home data:', homeData);
console.log('Rooms data:', response.items);
console.log('Room data:', roomData);

If you see these logs, your app is working but data might be wrong.

If you DON'T see these logs, there's a JavaScript error preventing execution.

Step 3: Check Network Tab

  1. In browser dev tools, go to "Network" tab
  2. Refresh the page
  3. Look for failed requests (red status codes)
  4. Check Contentful API calls (filter by "contentful")

💡 Common Fixes

If you see: "Cannot read property 'fields' of undefined"

Fix: Your Contentful entry structure doesn't match expected format

  • Go to Contentful and check your entry structure
  • Make sure you have a home entry with proper fields
  • Make sure your room entries have slug, price, images

If you see: "TypeError: Cannot read property 'map' of undefined"

Fix: The images field is not an array or doesn't exist

  • In Contentful, edit your room entry
  • Make sure images field is set to "Media" type, "multiple"
  • Add at least one image to the room

If console is completely empty

Fix: JavaScript bundle failed to load

  • Check if /static/js/bundle.js exists
  • Try clearing browser cache (Ctrl+Shift+R)
  • Restart dev server: npm start
